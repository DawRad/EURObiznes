Funkcja zwraca wartość, która pozwala określić dostępne dla tego pola akcje. 

W tym przypadku pozwala ona określić, że obiekt jest polem, na którym nie można budować nieruchomości, a czynsz zależy od liczby posiadanych przez właściciela pól tej klasy.
