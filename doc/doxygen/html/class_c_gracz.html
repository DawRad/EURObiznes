virtual int CGracz::decyduj ( int  stan,
int  kwota,
int  lbPol 
)
pure virtual

Funkcja zwracająca decyzję gracza dotycząca pola, na którym stanął.

Parametry
stan- stan pola, na którym stanął gracz:
  • 0 - wolne
  • 1 - zajęte
  • 2 - zastawione
kwota- jest to, w zależności od stanu pola, koszt jego kupna, budowy nieruchomości albo wykupienia z zastawu.
lbPol- parametr podawany w przypadku pól standardowych - mówi o liczbie posiadanych przez gracza pól koloru, jaki odpowiada polu, na którym gracz aktualnie się znajduje
Zwraca
Zwraca zgodę na wykonanie akcji (1) albo jej brak (0).

Implementowany w CCzlowiek i CKomputer.

◆ dodajPole()

int CGracz::dodajPole ( SWynAkcji  opt)

zwiększa o jeden wartość atrybutu CGracz wskazującego liczbę posiadanych przez gracza pól danego rodzaju

Parametry
optwskazuje, jaki rodzaj pola ma zostać dodany:
  • opt.y1 == 0 —> zwiększa wartość CGracz::lbPolSpec_
  • opt.y1 == 1 —> zwiększa wartość CGracz::lbPolKol_ (opt.y2 wskazuje, który kolor)
  • opt.y1 == 2 —> zwiększa wartość CGracz::lbPolEkstra_
