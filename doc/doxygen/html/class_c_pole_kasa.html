◆ akcja()

SWynAkcji CPoleKasa::akcja ( CGracz *  gracz)
virtual

Funkcja zwraca wartość, która pozwala określić dostępne dla tego pola akcje.

W tym przypadku pozwala ona określić, że obiekt jest polem, które dodaje lub odejmuje daną kwotę z zasobów gracza stającego.

Parametry
*gracz- wskaźnik do obiektu klasy gracz, który stanął na polu
Zwraca
Zwraca wartość SWynAkcji.y1 = 4 mówiąca, że to klasa CPoleKasa i SWynAkcji.y2 wskazującą, ile pieniędzy gracz otrzymuje albo traci.
