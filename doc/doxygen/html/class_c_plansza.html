static

Funkcja poszukuje indeksu początkowego i końcowego elementu typu string należącego do danego nagłówka.

Przeszukuje kontener deque<string> zawierający linijki danych i określa, który element jest elementem zawierającym zakończenie danego nagłówka ("/...").

Parametry
trescPlikuPodwójna lista składająca się z elementów typu string.
nazwaNagZawiera informację o nazwie poszukiwanego nagłówka (nie jego końca - </...>).
startIndeks elementu listy, od którego funkcja zaczyna analizę (powinien być to indeks początku nagłówka).
Zwraca
Zwraca numer elementu, który jest zakończeniem nagłówka.

SParsedElem CPlansza::parseElem ( string &  daneStr)
static

/brief Wyszukuje w strumieniu nagłówek i indeksy początku i końca jego zawartości.

Funkcja najpierw wyszukuje najbliższy nagłówek (<nagłówek>), a potem wyznacza indeks początkowy jego zawartości.

Parametry
daneStrStrumień danych (np. pobrany z pliku).
daneStrIndexIndeks znaku w strumieniu, od którego funkcja ma zacząć poszukiwać nagłówka.
Zwraca
Wartość zwracana to struktura zawierająca nazwę znalezionego nagłówka i numer pierwszego indeksu zawartości (pierwszego znaku po '>' początku nagłówka) oraz końca zawartości (ostatniego znaku przed '<' końca nagłówka).
