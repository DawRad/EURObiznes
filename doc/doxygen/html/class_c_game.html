◆ graczZmieniaPole()

void CGame::graczZmieniaPole ( int  lbKrokow)

Przemieszcza gracza na inne pole (poprzez zmianę następnej pozycji).

Metoda używana w przypadku, gdy gracz zmienia pole w wyniku akcji karty ruchu. W zależności od liczby kroków cofa się albo przesuwa naprzód.

Parametry
lbKrokow- wskazuje o ile pól zmienia się pozycja gracza (może być dodatnia - ruch do przodu - lub ujemna - ruch w tył)
